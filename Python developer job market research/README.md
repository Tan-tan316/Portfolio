# 🐍 Аналіз ринку праці Python-розробників (Stack Overflow 2025)

## 📌 Про проєкт
Цей аналіз базується на даних щорічного опитування **Stack Overflow за 2025 рік**. Мета дослідження — оцінити реальний попит на мову Python, вивчити демографію розробників та визначити ключові фактори, що впливають на рівень компенсації.

## 🛠 Технологічний стек
Для обробки та візуалізації даних використовувався **Jupyter Notebook** та наступні бібліотеки:
* **Pandas** — очищення, фільтрація та агрегація даних.
* **Matplotlib** — побудова графіків та візуалізація трендів.
